## Color Usage Guide

### Blue (Primary) — Use For:
- Headers and navigation bars
- Text links and interactive elements
- Form focus states and selections
- Secondary buttons (View, Learn More)
- Tab bar active state
- Skill level badges (Cat 3-6)

### Emerald (Secondary) — Use For:
- Primary CTAs (Book Now, Join Match, Confirm)
- Open match cards and highlights
- Availability indicators ("2 spots left")
- Price displays
- Court/venue type badges (Indoor, Outdoor)
- Positive action confirmations
