Onboard yourself on this codebase:

1. Read project structure, package.json, and config files
2. Read any existing CLAUDE.md, README, CONTRIBUTING docs
3. Read test setup and 2-3 recent test files for conventions
4. Read 2-3 recent source files for code patterns
5. Check git log (last 20 commits) for commit conventions
6. Run the test suite, typecheck, and lint
7. Summarize: stack, structure, patterns, health, conventions

If the codebase is large, use subagents to parallelize.
