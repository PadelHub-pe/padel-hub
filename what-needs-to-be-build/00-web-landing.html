📐 Landing Page — Spec Sheet

Page Purpose

  • B2B landing page for padel facility owners
  • Primary goal: drive "Solicitar Acceso" conversions
  • Secondary: educate on marketplace value prop
  • Route: / or /para-locales

Target Audience

  • ~30-50 padel facility owners in Lima
  • Facility managers / operations staff
  • Decision makers evaluating booking solutions

Page Sections

  • Navbar — Fixed, transparent → white on scroll
  • Hero — Value prop + dashboard mockup + floating cards
  • Social Proof — 4 key platform stats
  • Problem/Solution — Dark bg, 3v3 comparison
  • Features — 6 cards, 1 featured (2x2 grid)
  • How it Works — 3-step process
  • Comparison — PadelHub vs traditional software table
  • Testimonial — Single quote (placeholder)
  • CTA — Email capture + submit button
  • Footer — Minimal links

Key Messaging Hierarchy

  • H1: "Llena tus canchas con jugadores reales"
  • Subhead: Marketplace demand generation
  • Differentiator: We bring players TO you (vs. you managing yourself)
  • Trust: Beta activa, locales verificados, 24h review

CTA Strategy

  • Primary CTA: "Solicitar Acceso" (hero + footer)
  • Secondary CTA: "Ver cómo funciona" (scroll anchor)
  • MVP flow: Email → Manual review → Invite email
  • Post-MVP: Full "Solicitar Acceso" form (name, facility, district, courts, phone)

Design System

  • Display font: Sora (headings)
  • Body font: DM Sans (paragraphs, UI)
  • Primary: #3B82F6
  • Secondary: #10B981
  • Dark sections: #0F172A
  • Border radius: 12px (cards), 14px (buttons), 28px (large)

Technical Notes

  • Framework: Next.js 15 (same app as dashboard)
  • Route: Public route, no auth required
  • Navbar "Iniciar sesión" → /login
  • Email capture → trpc.accessRequest.create
  • Social proof stats: Static for MVP, dynamic later
  • Responsive: 3-col → 2-col → 1-col breakpoints

Post-MVP Enhancements

  • Replace email-only with full "Solicitar Acceso" form modal
  • Add video demo / Loom walkthrough
  • Real testimonials with photos
  • Dynamic stats from API
  • Localized SEO content per district
  • Blog / resources section
